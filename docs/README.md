## Documentação Rota Certa (v1)

Bem-vindo à documentação oficial do Rota Certa. Aqui você encontra as APIs públicas, SDKs, componentes de UI, esquemas de dados e guias de integração.

- Base URL de produção: `https://api.rotacerta.com/v1`
- Base URL de homologação: `https://staging.api.rotacerta.com/v1`
- Formato padrão: JSON (UTF-8)
- Auth: Bearer JWT (veja Autenticação)
